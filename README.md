# REDIS (simple Java project)

This repository contains a minimal Redis-like server implemented in Java.

## Goal
Upgrade project to run with Java 21 (LTS).

## Build with Maven
1. Install JDK 21 and Maven.
2. From the repository root run:

```powershell
mvn -v
mvn clean package
```
